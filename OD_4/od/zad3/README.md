normalne odpalenie poprzez komendę: (obrazek 1.png)
uwsgi --socket 127.0.0.1:29000 -w demo:app

uprawnienia root(uid == 0)

uwsgi --socket 127.0.0.1:29000 -w demo:app --uid 1000
